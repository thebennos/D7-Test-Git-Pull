Introduction
-----------
This module makes the current 
private message module to use
ajax submit.

Requirements
------------
Drupal 7.x
privatemsg 7.x

Installation
------------
1. Copy the entire ajax_privatemsg in
   the Drupal sites/all/modules directory.

2. Login as an administrator. 
   Enable the module in the "Administrator" -> "Modules"

How to use:
-----------
1.After installation it will automatically make forms ajax.
