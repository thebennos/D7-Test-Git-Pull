1. ABOUT
========

This module extends the ajax_facets module by adding support for:
- Current Search
- Search API Sorts

2. INSTALLATION
===============

Install as usual, see http://drupal.org/node/70151 for further information.
Contact me with any questions.

3. REQUIREMENTS
===============

This module requires:
Ajax Facets - https://www.drupal.org/project/ajax_facets
Libraries - https://www.drupal.org/project/libraries

This module supports:
Module Current Search - https://www.drupal.org/project/facetapi
Module Search API Sorts - https://www.drupal.org/project/search_api_sorts
